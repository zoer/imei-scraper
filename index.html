IMEI Scraper

Scrapes warranty infomation about Apple devices by IMEI number.

Installation

$ git clone https://github.com/zoer/imei-scraper
$ cd imei-sraper
$ bundle install

Usage

Use with a browser

Quick demo on heroku

or You can run it by self:

$ bundle exec rails s
$ open http://localhost:3000
